# 🚀 AI Resume Analyzer

A client-side web application that helps candidates optimize their resumes for Applicant Tracking Systems (ATS). By leveraging Google's Gemini AI and PDF.js, this tool extracts text directly from a user's uploaded PDF and compares it against a target job description.

## ✨ Features
* **Client-Side PDF Parsing:** Securely extracts text from resumes in the browser using `pdf.js`.
* **Skill Matching Engine:** Calculates a dynamic ATS compatibility score based on keyword overlap.
* **AI Career Insights:** Uses the Gemini 2.5 Flash API to generate personalized strengths, improvements, and recommended skills.
* **Responsive & Print-Ready:** Features a clean, SPA-style UI with native CSS print media queries for saving reports.

## 💻 Tech Stack
* HTML5 / CSS3
* Vanilla JavaScript (ES6+, Async/Await)
* Google Gemini API
* PDF.js & Marked.js
